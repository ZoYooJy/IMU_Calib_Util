  四、关键提取逻辑说明

  1. 白噪声（noise_density）：用 curve_fit 对 Allan 方差曲线的短时间部分（period ≤ 10s）拟合斜率为 −0.5 的直线，然后取该直线在 τ = 1 s 处的值（第 127–129 行、189–191
  行的大圆点标记）。
  2. 随机游走（random_walk）：对整条曲线拟合斜率为 +0.5 的直线，取该直线在 τ = 3 s 处的值（第 132–134 行、193–195 行的星号标记）。
  3. 零偏不稳定性（bias instability）：直接取 Allan 方差曲线的全局最小值（第 136–138 行），对应曲线趋平的区域。
  4. "最坏值"策略：YAML 输出中取三轴最大值（np.amax），这是 Kalibr 惯例——用最保守估计确保滤波器不会低估噪声。
  5. 陀螺仪单位转换：内部计算全程使用 deg，仅在写入 YAML 时乘以 π/180 转换为 rad（第 243–244 行）。
